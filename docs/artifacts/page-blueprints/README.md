# SKVN Gutenberg Page Blueprints

Status: reusable page starters.
Purpose: Gutenberg-safe starting points for online pages.

Unlike benchmark templates, these files are intended to be adapted into real site pages after human review.

Known render issues and full-width control notes are tracked in `../benchmark-templates/render-notes.md`.

Rules:

- Replace placeholder images with media-library assets.
- Keep content editable in Gutenberg.
- Do not add raw CSS or JavaScript.
- Do not include header/footer markup.
- Keep quote form behavior deferred until the approved milestone.

## Files

| File | Use |
|---|---|
| `seafood-export-home.gutenberg.html` | Homepage or campaign landing starter |
| `request-quote-intro.gutenberg.html` | Quote intro/request page visual surface for V1 / 0.6.0 |
